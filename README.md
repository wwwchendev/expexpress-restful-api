## Project Setup
1. 下載相依模組 `npm install`
2. [申請MongoDB資料庫、Node.js連線至mongoDB](https://wwwchen.dev/blogs/2jiC8H21jO7qaGFN1OHU)
3. 將`.env.template`改成`.env`並填上資料庫的連線代碼

## 運行伺服器
```
npm run start
```
